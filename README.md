## How to run the project

### Dependencies

- Install `deno >= 1.44`
- Install `node >= 20`
- Install `yarn`

### Setup

- Install deps with `yarn install`
- Install deps with `yarn install` at `app/client`
- Edit `app/server/config.yml` and set version to `development`

### Start project

- Run `deno task start`
